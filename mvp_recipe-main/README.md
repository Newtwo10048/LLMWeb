# 食譜營養計算 MVP 4.0

一個用 **Python + Flask + JSON** 打造的 **最小可行產品（MVP）**，  
讓使用者能從 **12 道高齡友善食譜** 中自由組合，  
一鍵計算 **1～50+ 人份** 的熱量、基礎營養份數、食材所需，並提供飲食建議。

**目前更新**：  
- 永遠顯示「一人份」營養（符合營養師慣例）  
- 食材 **中英對照**(如果有的話)（對接 Foodball / USDA 準備就緒）  
- 未來擴充 4000+ 道中文食譜（可能Cookpad 爬蟲）

## 專案結構（最新版）

```

├── app2.py                         ← MVP 3.0（一人份顯示）

├── taiwan_food_500_en_full.csv     ← 299 種常見食材中→英對照表(by Grok)

├── templates/
```

## 功能（MVP 4.0）

| 功能                 | 說明                                              |
|----------------------|---------------------------------------------------|
| 任意勾選 12 道食譜   | 高齡友善設計                                      |
| 0.5 ～ 50+ 人份調整  | 團膳專用                                          |
| 一人份熱量＆六大類   | 營養師最愛的呈現方式                              |
| 中英營養標示     | Vegetable (蔬菜)、Protein (豆魚蛋肉)…             |
| 食材中英翻譯(對照表)         | 高麗菜 → Cabbage、地瓜葉 → Sweet Potato Leaves   |
| 飲食建議             | 蛋白質、蔬菜、全穀不足自動提醒                    |
| 完整採購清單         | 含數量 + 單位，長照機構直接列印                   |
